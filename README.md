# HR Attrition Dashboard

## Project Overview

This project analyzes employee attrition using an interactive dashboard built in Google Sheets as part of the University of Nevada, Reno Data Analytics Program using the NCLab platform. The dashboard enables users to explore workforce trends and identify factors contributing to employee attrition.

## Dashboard Features

- Interactive drop-down filters
- KPI scorecards
- Department analysis
- Attrition by reason
- Employee demographics
- Dynamic charts and visualizations

## Tools Used

- Google Sheets
- Microsoft Excel
- Pivot Tables
- Dashboard Design
- Data Visualization

## Skills Demonstrated

- Dashboard Design
- Data Cleaning
- Pivot Tables
- Data Analysis
- Data Visualization
- KPI Reporting

## Dataset

Dataset: HR Attrition Dataset

Source: University of Nevada, Reno Data Analytics Program using the NCLab platform

## Key Insights

- Compared attrition trends across departments.
- Identified the primary reasons employees left the organization.
- Built interactive dashboards using filters and scorecards.
- Created dynamic visualizations to support business decision-making.
